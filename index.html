javascript入门

可插入HTML页面的轻量编程语言。特点：易学

Javascript 常见的使用方式有两种

第一种就是在html 文件中直接插入script，不过在项目中更多的是直接建立一个main.js 文件，然后在

html 中利用script引用该文件

然后在script 内部写执行代码。

Javascript 变量输出

  1. windows.alert(),弹出警告框

  2. document.write();将内容写到HTML文档中

  3. innerHTML 写入到innerHTML，使用如下

    <script>document.getElementById("demo").innerHTML = "段落已修改。";</script>

  4. console.log(),将内容写入到浏览器的控制台。

javascript语法

数字，字符串，bool类型，以及数组，对象，等都类似与c++；此外，几乎所有的变量都要使用

var来定义。var a=6; var name = 'sunny';此外，javascript每个语句后都不能缺少**;**

数组的定义

javascript 中还有一些其他关键字，注释也是，与c++类似；

函数定义

事件 描述
onchange HTML 元素改变
onclick 用户点击 HTML 元素
onmouseover 用户在一个HTML元素上移动鼠标
onmouseout 用户从一个HTML元素上移开鼠标
onkeydown 用户按下键盘按键
onload 浏览器已完成页面的加载

更多事件列表: JavaScript 参考手册 - HTML DOM 事件。

constructor 属性

constructor属性返回所有javascript 变量的构造函数，

类型转化

字符串转数字：Number("string")如果string是数字字符串，返回数字，如果不是，返回NaN

布尔值转数字：Number(“false");

日期转数字：d= new Date(); Number(d);或d = new Date();d.getTime()

数字转字符串：String(num) 或 num.toString()

日期转字符串;String(Date) 或 Date.toString()

正则表达式匹配规则

/正则表达式主体/修饰符 下面就是各种类型的修饰符及规则。

限定符

  • *（某个字符或字符串出现0次或多次）
  • 表示某个字符或字符串出现1次或多次+
  • ？表示某个字符或字符串出现0次或一次
  • a{6} 表示a出现6次
  • a{2,6} 表示a出现2-6次，一个范围
  • a{2,}表示a出现两次以上

或运算符与字符串

(a|b) 表示匹配a或b

[abc] 匹配a或b或c;

[a-z]小写字母

[0-9] 数字字符。

元字符

  • \d 匹配数字字符
  • \D 匹配非数字字符
  • \w 匹配单词字符（英文单词，下划线）
  • \s 匹配空白符
  • \S 匹配非空白符
  • . 匹配任意字符
  • ^a 匹配行首的a
  • a$ 匹配行尾的a

贪婪/懒惰匹配

<.+>默认贪婪匹配“任意字符”

<.+?>懒惰匹配任意字符。

a.经历过程

其实搭建，这个想法，我很早以前就有了。毕竟作为一个计算机人，有一个属于自己的博客网站就是一件很装B的事情。 这个事情，应该是想了一年都没有真正的实施。原因很简单，我觉得博客搭建太过高端，我也总感觉自己的实力达不到 什么都不会，又什么都会一点，所以就一直没有开始。当然了，也有害怕麻烦不愿意去做这件事情，最主要的原因还是 打心里害怕计算机，害怕失败。最近，突然发现身边的人，都好厉害，无论是学习还是技术，都好厉害。我突然觉得， 自己好弱小，突然觉得自己需要做些什么来进行改变。十月份刚开始的时候，我也是有想过利用Python Django来搭建 自己的网站，也确实是做到了一点点，至少在本地网站是可以看的，然后，我觉得后面都听不懂了，就没有继续学下去 因为在Django中涉及到一点点的HTML知识，大概过了一段时间，又开始学习HTML。就这样拖着拖着，就到了十月末，前 学的Django忘完了，HTML也就是会一点点，搭建自己的博客网站又得从零开始。真的有时候真是不太舒服。利用GitHub + hexo 搭建网站，看着教程，本以为就是无脑操作，没有想到就这种看着教程搞东西，还整整卡了两天，我真的是想 死的心都有了。问题一个接着一个，关键是好多我都解决不了。崩溃了一遍又一遍，不过还好，最终还是把这玩意搞出 来了，虽然很丑，很无语，不过吗，还是很有成就感的，因为，我能在网上搜到我的东西了。
