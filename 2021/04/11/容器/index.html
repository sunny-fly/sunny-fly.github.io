容器

vector 容器，c++常用的一种类型，相当于是可以用来存放任意类型的动态数组，方便代码简洁。

一般用法

vector<int>arr;像这种就是一个整型的动态数组。

此外,还有vector<string>s这种就是字符串型

vector<vector<int>>arr二维整型，还有像结构体等一些都是可以的。

初始化

  1. vector a；最常用的，定义一个空的容器。
  2. vector a(10)；定义一个初始长度为10的容器，并且，一般来说，这十个数都是0；
  3. vectora(10,1);初始长度为10,并且，每个数的初值都为1；
  4. vector a(b);直接将容器b复制到容器a中

常用操作

  • 向末尾插入新元素：arr.push_back(elem)向末尾压入元素，常用操作。
  • 向固定位置插入arr.insert(arr.begin()+i,new_elem)表示在第i个位置插入新元素，注意：这个必须以**arr.begin()**作为参考点。
  • 删除元素arr.erase(arr.begin()+i)删除第i个元素。
  • 容器长度: arr.size();

sort函数

可以对容器中的数进行排序，sort(a,b,way);三个参数，前面两个参数表示在(a,b)之间进行排序，并且不能包括b，第三个参数，表示的排序的方式,

如果没有第三个，默认是从小到大进行排序。
